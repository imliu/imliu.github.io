申请Github Pages 服务。

申请账户

邮箱

前往Github官网注册账户。
确认邮箱。
新建项目，项目名为“uesrname”.github.io.

本地环境搭建。

Github for windows 2.0

Node.js

下载Github for windows 在线安装包，自动安装。
在网页项目，点击Clone to desktop
选择下载路径。
下载Node.js安装包，安装。

配置Hexo

下载安装hexo

$ npm install -g hexo

安装hexo到指定文件夹

$ hexo init <folder>
$ cd <folder>
$ npm install

这样，博客搭建的环境就配置好了。
